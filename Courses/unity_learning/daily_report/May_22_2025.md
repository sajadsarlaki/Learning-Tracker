## âœ… What Is a Singleton?

A **Singleton** ensures that there's **only one instance** of a class (like `GameManager`) and allows any script to access it using `GameManager.Instance`.

```csharp
public static GameManager Instance;

void Awake()
{
    if (Instance != null)
        Destroy(gameObject);
    else
    {
        Instance = this;
        DontDestroyOnLoad(gameObject);
    }
}

```

Guarantees only one GameManager, accessible anywhere with `GameManager.Instance`.

### âœ… Why Singleton Rocks:

- ðŸ”„ Only one GameManager lives at a time.
- ðŸŒŽ Global access from any other script.
- â™»ï¸ Persists across scene loads via `DontDestroyOnLoad`.
- ðŸ’¡ No need to drag references manually all over the place.

3. A new field appears: **Render Camera**

4. **Important Trick**:
    - Make sure the **Canvas's "Plane Distance"** is > 1 (e.g., `100`)
    - And make sure the **Cameraâ€™s Culling Mask** includes the layer the Canvas is on.
    - You can either:
        - Set the Canvas to **layer 1**, or
        - Put it on the same layer your scene camera is rendering.
